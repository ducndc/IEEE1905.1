Parse a CMDU to get list of the TLVs of a single type

Parameters
[in]ccmdu_buff for parsing
[in|out]tv array of TLVs to hold the returned TLVs
[in]policypolicy for TLV parsing
[in|out]*num number of TLVs
Returns
0 on success, else non-zero.

This function can be used when number of TLVs of the same type is present more then 16 times; f.e. CMDUs containing TLVs for scanresults. The TLVs are returned in the passed 'tv' array. And, *num is updated with the actual number of TLVs present.
